# what-is-loop-engineering

An educational book on **Loop Engineering** — the practice of designing systems
that prompt, observe, verify, and iterate AI coding agents autonomously, rather
than prompting them manually turn-by-turn. It is written for software engineers
who already use AI coding agents but have not yet built systems to orchestrate
them.

Start reading at the book's table of contents: [book/README.md](book/README.md).
